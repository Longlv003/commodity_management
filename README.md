# commodity_management

**Dự án 1 (PRO1122)** — Quản lý hàng hóa  

---

## Giới thiệu
**Commodity Management** là ứng dụng giúp người dùng **quản lý danh mục hàng hóa**, bao gồm các chức năng:
- Thêm, sửa, xoá hàng hóa  
- Lưu trữ dữ liệu trên server (Node.js + MongoDB)  
- Đồng bộ dữ liệu giữa ứng dụng Android và server qua API  

Dự án gồm **2 phần chính**:
- **App (Android)**: Giao diện người dùng, phát triển bằng **Android Studio (Java)**  
- **Server (Node.js + Express + MongoDB)**: Xử lý API, quản lý dữ liệu hàng hóa
